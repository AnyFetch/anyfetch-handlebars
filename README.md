# Anyfetch-Handlebars
> Visit http://anyfetch.com for details about AnyFetch.

Inject new helpers in handlebars

# How to install?
```
npm install
```

Check everything is working with `npm test`.

# How to use it


```js
var hbs = require('anyfetch-handlebars');
hbs.loadAnyfetchHelpers(/* Your context */);
```
